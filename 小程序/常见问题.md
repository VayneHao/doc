### 小程序和传统 H5 的区别

| -        | H5                        | 小程序                      |
| :------- | :------------------------ | :-------------------------- |
| 运行环境 | 浏览器以及 APP 的 webview | 微信端                      |
| 语言     | JS、CSS、HTML             | WXML、WXCS                  |
| 组件     | 自行封装或第三方          | 微信内置                    |
| 协议     | http、https               | 只能是 https                |
| 权限     | 借助原生或高级语法        | 能直接调起如相册等          |
| 线程     | 单线程                    | 双线程 （渲染更快、更流畅） |
| 内核     | 浏览器等                  | 微信                        |
| DOM      | 有                        | 无 window、DOM 方法         |
